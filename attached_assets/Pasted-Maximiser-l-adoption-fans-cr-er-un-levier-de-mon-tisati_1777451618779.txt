Maximiser l’adoption (fans) + créer un levier de monétisation simple (prestataires & entreprises) sans friction.

⸻

👤 1. UTILISATEURS (FANS)

✅ Accès

* 100% gratuit
* Aucun paiement pour voir :
    * prestataires
    * trajets
    * événements
    * contacts

✅ Ils peuvent :

* voir les profils prestataires (photos, prix, description, WhatsApp)
* voir les trajets (ville, prix, places, conducteur)
* contacter directement (WhatsApp / tel / lien externe)

👉 Aucune barrière à l’entrée

⸻

🧑‍💼 2. PRESTATAIRES (COIFFEUR, MAKEUP, ETC.)

🆓 Offre gratuite (par défaut)

* création de fiche
* infos :
    * nom
    * ville
    * catégorie
    * description
    * 3 photos
    * contact (WhatsApp)

💰 Options payantes (monétisation principale)

🔥 Boost visibilité

* Boost 24h → 5€
* Boost événement (week-end concert) → 15€
* Top prestataire (ville/catégorie) → 25€

✅ Badge

* Badge vérifié → 10€

📦 Pack

* Pack visibilité complet → 49€

👉 Logique : ils payent pour être visibles, pas pour exister

⸻

🏢 3. ENTREPRISES (MARQUES / BUSINESS / PARTENAIRES)

❗ Toujours payant (pas de gratuit)

Types :

* marques (boissons, mode, etc.)
* restaurants
* clubs / after-party
* agences
* sponsors

💰 Offres

📍 Placement sponsorisé

* Bannière ou mise en avant → prix fixe

🔥 Fiche sponsorisée

* profil mis en avant dans les listes
* tag “sponsorisé”

📲 Contenu sponsorisé

* post mis en avant dans feed
* visibilité sur page événement

📦 Packs entreprise

* pack event complet (ex : concert Fally)
* visibilité globale plateforme

👉 Logique : les entreprises payent pour l’audience

⸻

🚗 4. COVOITURAGE

🆓 Gratuit pour publier

* conducteur peut publier librement

⚠️ Pas de paiement via plateforme (V1)

* contact direct (WhatsApp)

👉 Éviter :

* gestion paiement
* litiges
* remboursement

⸻
